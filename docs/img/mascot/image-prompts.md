# Image Generation Prompts for Buzz the Honey Bee

Buzz is the pedagogical mascot for the *Networking and Communication* textbook. Bees use the **waggle dance** to communicate location and distance to other foragers — a perfect biological analog to network protocols.

Use the prompts below in any AI image generator (ChatGPT/DALL-E, Midjourney, Stable Diffusion, etc.). Each prompt is **fully self-contained** — paste any one into a fresh chat and you should get a consistent Buzz pose.

After generating each image:

1. Save it as the indicated filename in `docs/img/mascot/`
2. Trim transparent padding by running:
   ```bash
   python $BK_HOME/src/image-utils/trim-padding-from-image.py docs/img/mascot/<FILENAME>.png
   ```

Required output: PNG, 512×512 or 1024×1024, fully transparent background, no text in image.
